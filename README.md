## Установка и запуск проекта

```bash
# 1. Клонирование репозитория
git clone <URL_вашего_репозитория>

# 2. Переход в директорию проекта
cd recruting-app

# 3. Установка зависимостей
pnpm i

# 4. Запуск фронтенда
pnpm run dev

# 5. Запуск бэкенда
pnpm run server
